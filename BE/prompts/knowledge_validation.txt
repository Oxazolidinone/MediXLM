Medical Knowledge Validation Prompt

Knowledge to Validate:
- Name: {name}
- Type: {knowledge_type}
- Description: {description}
- Source: {source}
- Properties: {properties}

Validation Criteria:
1. Medical Accuracy
   - Verify terminology is correct
   - Check for medical contradictions
   - Validate against established guidelines

2. Completeness
   - Essential properties are present
   - Description is comprehensive
   - Source is credible

3. Consistency
   - Knowledge type matches content
   - Properties align with type
   - No conflicting information

4. Clinical Relevance
   - Applicable in medical context
   - Evidence-based information
   - Current medical standards

Output:
- Validation status (pass/fail)
- Confidence score (0-1)
- Issues found
- Suggestions for improvement
